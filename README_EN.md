## Professional profile

I am a product-oriented software engineer with a strong focus on **backend development, APIs, modular architecture and business logic**. My profile combines full-stack delivery capability with especially strong value in service design, domain structure and maintainable technical foundations.

I work with a clear mindset:

- build useful software, not only attractive interfaces
- keep codebases clean and easy to evolve
- connect frontend, backend and database with strong technical judgment
- deliver complete features with attention to quality and clarity

I am especially interested in **Backend Developer** and **Software Engineer with backend focus** roles, while remaining comfortable in full-stack environments when the product requires end-to-end ownership.

## What I bring

- Design and implementation of clear, maintainable **REST APIs**

- Backend development with **NestJS, Node.js, Java and Python**

- Solid experience connecting Angular frontends with robust backend services
- Product-oriented execution with attention to maintainability and delivery quality
