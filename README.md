## Perfil profesional

Soy ingeniero de software orientado a producto, con especial foco en **backend, APIs, arquitectura modular y logica de negocio**. Mi perfil combina capacidad full-stack con una base especialmente fuerte para construir servicios, estructurar dominios y desarrollar aplicaciones que resuelven problemas reales de negocio.

Trabajo con una mentalidad muy concreta:

- construir software util, no solo interfaces atractivas
- mantener una estructura limpia y facil de evolucionar
- conectar frontend, backend y base de datos con criterio tecnico
- entregar funcionalidades completas con foco en calidad y claridad

Actualmente estoy especialmente interesado en oportunidades como **Backend Developer** o **Software Engineer con foco backend**, aunque puedo aportar valor real en entornos full-stack cuando el producto lo necesita.

## Lo que aporto a un equipo

- Diseno e implementacion de **APIs REST** claras y mantenibles
- Arquitectura modular por dominios y separacion de responsabilidades
- Backend con **NestJS, Node.js, Java y Python**

- Experiencia conectando frontend Angular con servicios backend robustos
- Mentalidad orientada a producto, mantenibilidad y ejecucion

## Proyecto destacado: I Do Manager

**I Do Manager** es una plataforma SaaS modular orientada a wedding planners y clientes finales. Este proyecto representa muy bien mi forma de trabajar porque no esta planteado como una demo aislada, sino como un producto con estructura real, dominio claro y recorrido tecnico.

### Que demuestra este proyecto

- Arquitectura full-stack con **Angular + NestJS + Prisma + PostgreSQL**
- Backend organizado por entidades y dominios de negocio
- Autenticacion, eventos, plantillas y documentos con estructura mantenible
- Modelo de datos claro con `User`, `Business`, `Event`, `Template` y `Document`
- APIs versionadas y base preparada para escenarios SaaS
- Enfoque de producto real: dashboard, editor, timeline, marketplace y flujos operativos

### Por que es relevante para recruiters tecnicos

- Refleja capacidad para construir software de negocio mas alla de ejercicios academicos
- Enseña estructura backend solida y evolucionable
- Muestra pensamiento de arquitectura, no solo implementacion puntual
